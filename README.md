# OptiMarket

OptiMarket is a Streamlit application that simulates a bond portfolio optimization tool. It uses live US Treasury rates to generate a synthetic market of corporate bonds and then finds the optimal portfolio to match a user's risk (duration) and maximize yield.

## How to Run

1.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the Streamlit app:
    ```bash
    streamlit run app.py
    ```
